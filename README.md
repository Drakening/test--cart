Shopping Cart Project

This is a simple multi-page shopping cart application built using HTML, CSS, and JavaScript. The project includes features like browsing products, adding them to the cart, updating quantities, and viewing a checkout page.

Features

Multi-page Navigation: Separate pages for Home, Products, Cart, and Checkout.

Add to Cart: Users can add items to the cart from the product list.

Cart Management: View items in the cart, update quantities, and remove items.

Responsive Design: Fully responsive layout for seamless viewing on different devices.

Persistent Data: Cart data is stored in local storage, so it persists across page reloads.
